# My Shopping List

This is a simple Shopping list projects that entails HTML, CSS and JS.

## Requirements
- [x] Array for storing shopping items
- [x] Event Listener for *add* and *clear list* buttons
- [x] Event Listener for *checkbox*
- [x] Calculating cost of marked items
- [ ] Local storage for array persistence.
- [ ] Seperate cards for items during rendering
- [ ] Editing and deleting items seperately
